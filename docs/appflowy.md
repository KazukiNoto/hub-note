# AppFlowy — Notionのオープンソース代替ツール

**Source**: <https://github.com/AppFlowy-IO/AppFlowy>

## 概要

**AppFlowy** は「Notionのオープンソース代替」として開発されたAIワークスペースです。プロジェクト管理、Wiki、チームコラボレーションをAI統合とともに提供しつつ、**データのコントロールを失わない**ことを重視した設計になっています。

GitHubスター数は約 **69,600**（2026年4月時点）で、オープンソースの生産性ツールとして高い人気を誇ります。

## 主な機能

- **カンバンボード** — タスク管理をビジュアルに整理
- **データベース機能** — プロジェクトや情報を構造化して管理
- **サイト作成** — ドキュメントやWikiを構築・公開
- **AI統合** — AIアシスタント機能を内蔵
- **テンプレートシステム** — すぐに使えるテンプレートを多数用意
- **クロスプラットフォーム** — デスクトップ（Windows・macOS・Linux）およびモバイル（iOS・Android）に対応
- **セルフホスティング対応** — 自社サーバーでのデプロイが可能

## 技術スタック

| 言語 | 割合 | 用途 |
|------|------|------|
| Dart / Flutter | 73.8% | フロントエンド（UI） |
| Rust | 24.1% | バックエンド（コアロジック） |
| C++, Shell, CMake | 残り | ビルドシステムほか |

Flutter と Rust の組み合わせにより、ネイティブに近いパフォーマンスとクロスプラットフォーム対応を両立しています。

## インストール・導入方法

=== "デスクトップ"
    - **macOS / Windows / Linux**: GitHub Releases からインストーラをダウンロード
    - **Linux**: FlatHub または Snapcraft 経由でも入手可能
    - **SourceForge** にも配布あり

=== "モバイル"
    - **iOS**: App Store（iPhone 対応）
    - **Android**: Google Play Store（Android 10 以上）

=== "セルフホスト"
    - 公式ドキュメントに OS 別のセットアップ手順あり
    - 自社環境へのデプロイで完全なデータ管理が可能

## プロジェクトの理念

開発者たちは既存ツール（特にNotion）の**セキュリティやモバイル対応の限界**を感じ、AppFlowy を立ち上げました。プロジェクトは以下の3つの価値観を基盤としています：

!!! note "コアバリュー"
    1. **データプライバシー** — ユーザーが自分のデータを完全にコントロールできる
    2. **信頼性の高いネイティブ体験** — ネイティブアプリとしての高いパフォーマンス
    3. **コミュニティ主導の拡張性** — オープンソースでコミュニティが機能を育てる

## まとめ

AppFlowy は Notion の有力なオープンソース代替として、プライバシーを重視するユーザーや自社サーバーで運用したい組織に適しています。Flutter + Rust という現代的な技術スタックにより、クロスプラットフォームでのネイティブ体験を提供しており、活発なコミュニティとともに急速に機能が拡充されています。
