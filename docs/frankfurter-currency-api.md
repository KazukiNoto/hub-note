# Frankfurter - 無料為替レートAPI

**Source**: <https://frankfurter.dev/>

## 概要

Frankfurterは、オープンソースの通貨データAPIで、54の中央銀行から毎日の為替レートを追跡しています。1948年からの164の現在アクティブな通貨と34の歴史的通貨に対応しており、認証なしで無料で利用できます。

## 主な特徴

- **無料・認証不要** — APIキーは不要
- **使用制限なし** — レート制限やクォータなし
- **毎営業日更新** — 16:00 CET頃に更新
- **歴史的データ対応** — 1948年から現在まで
- **オープンソース** — GitHub上で公開

## APIバージョン

- **v2 API** — 現在のバージョン（https://frankfurter.dev/）
- **v1 API** — 初期バージョン（https://frankfurter.dev/v1/）

## 主なエンドポイント

APIはapi.frankfurter.devでホストされており、以下の機能を提供します:

- **最新の為替レート** — 現在のレートを取得
- **ベース通貨変更** — `base`パラメータで基準通貨を指定
- **特定通貨のフィルタリング** — `quotes`パラメータで対象通貨を選択
- **過去のレート取得** — 特定日付の為替レートを検索
- **期間範囲検索** — `from`と`to`パラメータで期間指定

## データソース

通貨データは以下から取得されます:

- 54の中央銀行からの制度的なソース
- 164のアクティブな通貨
- 34の歴史的通貨
