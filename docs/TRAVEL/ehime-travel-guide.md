# 愛媛県 旅行ガイド — 観光地・グルメ・モデルコース

## 概要

愛媛県は四国の北西部に位置し、日本最古の温泉「道後温泉」やサイクリストの聖地「しまなみ海道」、現存12天守のひとつ「松山城」など、歴史・自然・グルメが揃った魅力的な観光エリア。中予（松山）・東予（今治）・南予（宇和島・大洲）の3エリアに分かれ、それぞれ個性豊かなスポットが点在している。

---

## おすすめ観光スポット

### 中予エリア（松山市周辺）

| スポット | 概要 |
|---|---|
| **道後温泉本館** | 日本最古といわれる温泉。夏目漱石『坊っちゃん』にも登場した愛媛のシンボル |
| **松山城** | 現存12天守のひとつ。天守閣から松山市街と瀬戸内海を一望できる |
| **道後温泉別館 飛鳥乃湯泉** | 飛鳥時代をイメージした新しい温泉施設。アート作品も楽しめる |
| **坊っちゃんカラクリ時計** | 道後温泉駅前のからくり時計。定時になると『坊っちゃん』のキャラクターが登場 |

### 東予エリア（今治・しまなみ海道）

| スポット | 概要 |
|---|---|
| **しまなみ海道** | 全長約60kmの海の道。サイクリングロード併設で「サイクリストの聖地」として世界的に有名 |
| **今治城** | 藤堂高虎が築いた日本屈指の海城。堀に海水を引き入れた珍しい構造 |
| **タオル美術館** | 今治タオルの製造工程見学やタオルアート展示が楽しめる |
| **亀老山展望公園** | しまなみ海道随一の絶景スポット。来島海峡大橋と瀬戸内海を一望 |

### 南予エリア（大洲・宇和島・内子）

| スポット | 概要 |
|---|---|
| **下灘駅** | 「一度は降りてみたい駅」として有名。ホームから瀬戸内海が広がる絶景駅 |
| **大洲城** | 4層4階の木造復元天守。肱川沿いの美しいロケーション |
| **内子の町並み** | 江戸〜明治の商家が残る重要伝統的建造物群保存地区。木蝋で栄えた歴史ある町 |
| **宇和島城** | 現存12天守のひとつ。伊達家ゆかりの城で天守からの眺望が素晴らしい |
| **四国カルスト** | 「天空の道」と呼ばれる標高約1,400mの石灰岩台地。爽快なドライブが楽しめる |

### その他の注目スポット

| スポット | 概要 |
|---|---|
| **伊予灘ものがたり** | レトロモダンな観光列車。海を眺めながら地元食材の料理を堪能できる |
| **石鎚山** | 西日本最高峰（標高1,982m）。四国を代表する霊山で登山・紅葉の名所 |

## おすすめご当地グルメ

### 鯛めし（2種類）

愛媛の県魚「鯛」を使った鉄板グルメ。地域で2つのスタイルがある。

| 種類 | 特徴 |
|---|---|
| **松山風（炊き込み）** | 鯛をまるごと一匹、米と一緒に炊き上げる豪快な炊き込みご飯 |
| **宇和島風（漬け丼）** | 鯛の刺身を醤油ベースのだし汁と生卵に絡ませ、熱々のご飯にのせて食べる漁師料理 |

### 麺・主食

| グルメ | 特徴 |
|---|---|
| **鍋焼きうどん** | 松山名物。アルミ鍋で提供される、ほのかに甘いやさしい味わいのうどん |
| **八幡浜ちゃんぽん** | 魚介のうまみたっぷり。長崎ちゃんぽんとは異なるあっさり系 |
| **焼豚玉子飯** | 今治のB級グルメ。ご飯にチャーシュー＋半熟目玉焼き＋甘辛タレの丼 |

### 肉・海鮮

| グルメ | 特徴 |
|---|---|
| **じゃこ天** | ホタルジャコを骨・皮ごとすりつぶして揚げた南予の名産品。カルシウムたっぷり |
| **今治焼き鳥** | 鉄板で上からプレスして焼くのが今治スタイル。皮がパリパリ |
| **三津浜焼き** | 大正時代から続く松山のお好み焼き。クレープ状の生地が特徴 |
| **宇和島真珠** | 食べ物ではないが、宇和島は真珠養殖で有名。お土産にも人気 |

### スイーツ・特産品

| グルメ | 特徴 |
|---|---|
| **愛媛みかん** | 全国トップクラスの生産量。ポンジュースでもおなじみ |
| **坊ちゃん団子** | 漱石の小説にちなんだ三色団子。抹茶・黄身・小豆の3色 |
| **一六タルト** | ゆず風味の餡をカステラ生地で巻いた愛媛の銘菓 |
| **母恵夢（ポエム）** | バター風味の皮に白餡を包んだ洋風まんじゅう |

## 1泊2日モデルコース

### プランA：松山・道後 定番満喫コース

**1日目（松山市内〜道後温泉）**

1. 🏯 松山城をロープウェイで登城。天守閣から絶景を楽しむ
2. 🍜 昼食：鍋焼きうどん or 三津浜焼き
3. 🚃 坊っちゃん列車で道後温泉へ移動
4. ♨️ 道後温泉本館 or 飛鳥乃湯泉で入浴
5. 🏘️ 道後温泉商店街を散策（坊ちゃん団子・一六タルト）
6. 🐟 夕食：松山風鯛めし
7. 🏨 道後温泉の旅館で宿泊

**2日目（内子〜下灘）**

1. 🏘️ 内子の町並みを散策
2. 🍊 昼食：じゃこ天＆みかんジュース
3. 🚉 下灘駅で瀬戸内海の絶景を堪能
4. 🏔️ 時間があれば大洲城を見学

### プランB：しまなみ海道サイクリングコース

**1日目（今治〜しまなみ海道）**

1. 🚲 今治からしまなみ海道をサイクリング（来島海峡大橋〜大島〜伯方島）
2. 🌊 亀老山展望公園で絶景を楽しむ
3. 🍗 昼食：焼豚玉子飯 or 今治焼き鳥
4. 🏯 今治城を見学
5. ♨️ 今治の温泉で疲れを癒す
6. 🏨 今治市内で宿泊

**2日目（松山〜道後温泉）**

1. 🏯 松山城を散策
2. 🐟 昼食：宇和島風鯛めし
3. ♨️ 道後温泉で入浴
4. 🛍️ 道後温泉商店街でお土産購入

    松山市内は路面電車（伊予鉄道）が便利。しまなみ海道や南予エリアはレンタカーがおすすめ。しまなみ海道のサイクリングはレンタサイクルターミナルが各島にあり、乗り捨てOK。

## 無料キャンプ場

愛媛県は山・海の自然が豊富で、無料または格安で利用できるキャンプ場が多い。水道・トイレ完備の施設も多く、気軽にアウトドアを楽しめる。

### おすすめ無料・格安キャンプ場

| キャンプ場 | エリア | 特徴 | 料金 |
|---|---|---|---|
| **姫鶴平キャンプ場** | 久万高原町（四国カルスト） | 標高約1,400mの高原。牛の放牧風景、満天の星空、パノラマの絶景が魅力。車中泊も可能 | 無料 |
| **北条鹿島キャンプ場** | 松山市（鹿島） | 海上の島にあるキャンプ場。海水浴・釣り・鹿のエサやりも楽しめる。炊事棟・水洗トイレ完備 | 無料 |
| **龍岡キャンプ場** | 今治市 | 今治ICから約12km。国道沿いでアクセス良好 | 無料 |
| **須ノ川公園キャンプ場** | 愛南町 | 足摺宇和海国立公園内。「日本の渚100選」の海岸でシュノーケリングやシーカヤック体験も可能 | 清掃協力費300円/人 |

!!! warning "キャンプ場利用時の注意"
    - 無料キャンプ場でもマナーを守り、ゴミは持ち帰ること
    - 姫鶴平は標高が高く夏でも夜間は冷え込むため防寒対策が必要
    - 事前に公式サイトで最新の利用条件・開設期間を確認すること

## 車中泊スポット（道の駅・RVパーク）

愛媛県には車中泊に適した道の駅やRVパークが点在。温泉併設や絶景が楽しめるスポットも多い。

### おすすめ車中泊スポット

| スポット | 所在地 | 特徴 |
|---|---|---|
| **道の駅ふたみ** | 伊予市双海町 | 瀬戸内屈指の夕陽スポット。海に沈む夕日が絶景。恋人の聖地としても有名 |
| **RVパーク 八幡浜みなっと** | 八幡浜市 | フェリーターミナル隣接。年間100万人が訪れる観光スポット。みかん畑と海の景色が目の前に広がる |
| **RVパーク 清流の里ひじかわ** | 大洲市 | 道の駅併設。肱川沿いの静かな環境でゆったり過ごせる |
| **道の駅マイントピア別子** | 新居浜市 | 大型駐車場完備（普通車381台）。別子銅山の歴史テーマパーク併設。温泉施設あり |
| **道の駅よしうみいきいき館** | 今治市（大島） | しまなみ海道沿い。来島海峡大橋の絶景と新鮮な海鮮バーベキューが人気 |

!!! tip "車中泊のマナー"
    道の駅は公共の休憩施設であり、宿泊施設ではない点に注意。長期滞在・テントの設営・外での調理は避け、エンジンのかけっぱなしにも配慮すること。RVパークは車中泊を正式に許可された施設なので安心して利用できる。
