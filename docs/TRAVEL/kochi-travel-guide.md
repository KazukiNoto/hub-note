# 高知県 旅行ガイド — 観光地・グルメ・モデルコース

## 概要

高知県は四国の南側に位置し、太平洋に面した雄大な自然と豊かな食文化が魅力のエリア。「最後の清流」四万十川、四国最南端の足摺岬、坂本龍馬ゆかりの桂浜など、スケールの大きな絶景が楽しめる。森林率日本一の山の国でもあり、海・川・山すべてが揃った「食材の宝庫」として知られている。

## おすすめ観光スポット

### 高知市エリア

| スポット | 概要 |
|---|---|
| **高知城** | 天守と御殿がセットで現存する日本唯一の城。本丸��建造物がほぼ完全な形で残る貴重な名城 |
| **桂浜** | 月の名所として「よさこい節」にも唄われる高知を代表する景勝地。龍頭岬に坂本龍馬の銅像が立つ |
| **ひろめ市場** | 約60店舗が集まる屋台村スタイルの市場。カツオのたたきやうつぼ料理など高知グルメを一堂に楽しめる |
| **日曜市** | 300年以上の歴史がある街路市。約1.3kmにわたり新鮮な野菜・果物・特産品の露店が並ぶ |
| **龍河洞** | 日本三大鍾乳洞のひとつ。約1kmの観光コースで約40分の神秘的な洞窟体験 |

### 四万十・足摺エリア

| スポット | 概要 |
|---|---|
| **四万十川** | 「日本最後の清流」。ダムのない自然のままの美しい流れ。カヌーや遊覧船も楽しめる |
| **佐田沈下橋** | 四万十川最下流にある最長の沈下橋。増水時に水面下に沈む欄干のない橋が独特の風景を生む |
| **足摺岬** | 四国最南端の岬。黒潮が直接ぶつかる日本唯一の場所。断崖の灯台からの眺望は圧巻 |
| **柏島** | 透明度抜群の海。船が宙に浮いて見えるほどの美しさで「高知のモルディブ」とも呼ばれる |
| **竜串海域公園** | 奇岩と美しいサンゴが広がる海中公園。グラスボートで海中散歩が楽しめる |

## おすすめご当地グルメ

### 海の幸

| グルメ | 特徴 |
|---|---|
| **鰹のたたき** | 高知の代名詞。表面を藁焼きで炙り、にんにく・生姜・ネギをたっぷりのせて食べる。地元では「塩たたき」が通の食べ方 |
| **うつぼのたたき** | 「海のギャング」うつぼの意外な美味しさ。皮下のゼラチン質がプルプルで濃厚な旨み |
| **清水サバ** | 土佐清水市で水揚げされるゴマサバ。鮮度抜群で刺身で食べられる |
| **室戸キンメ丼** | 室戸産キンメダイの照焼きと地魚の刺身盛り合わせ。しっとり食感が絶品 |

### 郷土料理

| グルメ | 特徴 |
|---|---|
| **皿鉢料理（さわちりょうり）** | 大皿に刺身・寿司・揚げ物などを豪快に盛り付ける土佐の宴会料理様式 |
| **鍋焼きラーメン** | 須崎市のご当地ラーメン。土鍋で提供される熱々の鶏ガラスープが特徴 |
| **屋台餃子** | 高知市の夜の名物。薄い皮でたっぷりの油でパリパリに揚げ焼き。小ぶりで何個でも食べられる |
| **芋けんぴ** | 高知発祥のさつまいもスナック。カリカリ食感と甘さがやみつきに |

### 肉・その他

| グルメ | 特徴 |
|---|---|
| **土佐ジロー** | 土佐地鶏とロードアイランドレッドの掛け合わせ。放し飼いで育てられ、肉も卵も味わい濃厚 |
| **四万十の天然鮎** | 清流で育った天然鮎の塩焼き。夏の風物詩 |
| **四万十の川海老** | 四万十川で獲れる手長海老の唐揚げ。サクサク食感で酒のつまみに最高 |

### お酒・ドリンク

| グルメ | 特徴 |
|---|---|
| **土佐の地酒** | 高知は「酒国土佐」と呼ばれる日本有数の酒どころ。辛口でキレのある淡麗な日本酒が多い |
| **ゆず** | 高知県は日本一のゆず生産地。ゆずジュース・ゆずポン酢など特産品多数 |

## 1泊2日モデルコース

### プランA：高知市＆桂浜 王道コース

**1日目（高知市内）**

1. 🛍️ 日曜市で朝の散策（日曜のみ開催）
2. 🏯 高知城で天守閣と御殿を見学
3. 🐟 昼食：ひろめ市場で鰹のたたき＆屋台餃子
4. 🌊 桂浜で坂本龍馬像と太平洋の絶景を堪能
5. 🐬 桂浜水族館を見学
6. 🍶 夕食：地元の居酒屋で皿鉢料理と土佐の地酒
7. 🏨 高知市内で宿泊

**2日目（龍河洞〜仁淀川）**

1. 🕳️ 龍河洞で約40分の鍾乳洞探検
2. 🎨 北川村「モネの庭」マルモッタンで庭園散策
3. 🍜 昼食：須崎市で鍋焼きラーメン
4. 💎 仁淀川エリアで仁淀ブルーを体感（にこ淵 or 安居渓谷）

### プランB：四万十川＆足摺岬 大自然コース

**1日目（四万十川）**

1. 🛒 久礼大正町市場で新鮮な魚介を堪能
2. 🏭 海洋堂ホビー館四万十でフィギュアの世界を楽しむ
3. 🛶 四万十川でカヌー or 遊覧船体験
4. 🌉 佐田沈下橋を散策
5. ♨️ 足摺温泉郷で宿泊

**2日目（足摺岬〜柏島）**

1. 🗼 足摺岬で太平洋の大パノラマを堪能
2. 📖 ジョン万次郎資料館を見学
3. 🐠 竜串海域公園でグラスボート体験
4. 🏝️ 柏島で透明度抜群の海を楽しむ
5. 🍽️ 昼食：土佐清水市で清水サバ

!!! tip "四万十・足摺方面はレンタカー必須"
    四万十川や足摺岬エリアは公共交通が限られるため、レンタカーでの移動が基本。高知市内からは車で約2〜3時間かかるため、余裕を持ったスケジュールを。

## 車中泊スポット（道の駅・RVパーク）

高知県は東西に長く、ドライブ旅行に適した道の駅が充実。温泉やグルメ施設を備えたスポットも多い。

### おすすめ車中泊スポット

| スポット | 所在地 | 特徴 |
|---|---|---|
| **道の駅 やす** | 香南市 | 高知市内から約40分。太平洋を望む好立地。室戸方面への中継地として便利 |
| **道の駅 なかとさ** | 中土佐町 | 高知市内から約50分。新鮮な海鮮が楽しめるレストラン併設。四万十方面への拠点に |
| **道の駅 南国風良里** | 南国市 | 南国ICから約3分。高知県の玄関口。農産物直売所・カフェレストラン併設 |
| **道の駅 すくもサニーサイドパーク** | 宿毛市 | RVパーク併設（1泊1,500〜2,000円）。キャンプ場もあり車中泊に最適 |
| **道の駅 四万十大正** | 四万十町 | 四万十川の中流域。鮎やうなぎなど川の幸グルメが楽しめる |

!!! tip "高知市内での車中泊"
    高知市中心部で車中泊するなら「天神橋パーキング」が便利。ひろめ市場まで徒歩圏内で、高知城や日曜市にもアクセスしやすい。
