# 香川県 旅行ガイド — 観光地・グルメ・モデルコース

## 概要

香川県は四国の北東部に位置する日本最小の県ながら、讃岐うどん・瀬戸内海の絶景・アートの島々・歴史ある神社仏閣など見どころが凝縮されたエリア。「うどん県」の愛称で親しまれ、コンパクトな県土のおかげで1泊2日でも主要スポットを効率よく巡れるのが魅力。

---

## おすすめ観光スポット

### 高松エリア

| スポット | 概要 |
|---|---|
| **栗林公園** | 国の特別名勝。紫雲山を背景にした回遊式大名庭園で、四季折々の美しさが楽しめる |
| **屋島** | 源平合戦の古戦場。山上から瀬戸内海の多島美を一望できる。新屋島水族館や四国村も人気 |
| **高松港・サンポート高松** | 直島や小豆島へのフェリー発着地。海沿いの開放的なウォーターフロント |

### 琴平・丸亀エリア

| スポット | 概要 |
|---|---|
| **金刀比羅宮（こんぴらさん）** | 785段の石段を登って参拝する四国屈指のパワースポット。海上守護の神様として有名 |
| **丸亀城** | 現存12天守のひとつ。日本一高い石垣（約60m）が見事 |
| **総本山善通寺** | 弘法大師空海の誕生地。四国八十八ヶ所霊場の第75番札所 |
| **中野うどん学校** | 讃岐うどんの手打ち体験ができる人気スポット |

### 西讃エリア（観音寺・三豊）

| スポット | 概要 |
|---|---|
| **父母ヶ浜（ちちぶがはま）** | 約1kmのロングビーチ。干潮時に水面が鏡のように反射し「日本のウユニ塩湖」と話題のフォトスポット |
| **銭形砂絵** | 有明浜に描かれた巨大な寛永通宝の砂絵。見た人はお金に不自由しないという伝説がある |
| **紫雲出山（しうでやま）** | 瀬戸内海を360度見渡せる絶景スポット。春の桜と海のコラボが人気 |
| **豊稔池堰堤** | 日本最古の石積みアーチダム。中世ヨーロッパの城壁を思わせる重厚な外観 |

### 離島エリア

| スポット | 概要 |
|---|---|
| **直島** | 世界的に有名なアートの島。地中美術館、ベネッセハウス、草間彌生の黄色いかぼちゃなど現代アートの宝庫 |
| **小豆島** | オリーブの島。寒霞渓（日本三大渓谷）、エンジェルロード（恋人の聖地）、二十四の瞳映画村など見どころ多数 |
| **豊島（てしま）** | 豊島美術館が有名。棚田と瀬戸内海の風景が美しい静かなアートの島 |
| **男木島・女木島** | 瀬戸内国際芸術祭の会場。猫の島としても人気の男木島、鬼ヶ島伝説の女木島 |

## おすすめご当地グルメ

### 麺・主食

| グルメ | 特徴 |
|---|---|
| **讃岐うどん** | 香川の代名詞。コシの強い麺が特徴で、かけ・ぶっかけ・釜揚げ・釜玉など多彩なスタイルがある。県内に600店舗以上 |
| **あんもち雑煮** | 白味噌仕立てのだしに甘いあんもちを入れる香川独特のお雑煮。意外な甘じょっぱさがクセになる |
| **しっぽくうどん** | 根菜や鶏肉を煮込んだ具だくさんの温かいうどん。冬の定番 |

### 肉・海鮮

| グルメ | 特徴 |
|---|---|
| **骨付鳥** | 丸亀市発祥。鶏もも肉をニンニク・黒胡椒・唐辛子でスパイシーに味付けしてオーブンで焼き上げる。ビールとの相性抜群 |
| **瀬戸内の魚介** | 春のサワラ、夏のマナガツオ、秋のチヌ、冬のハマチなど四季折々の新鮮な海の幸 |
| **オリーブハマチ** | オリーブの葉を混ぜた餌で育てた小豆島のブランドハマチ。さっぱりとした脂が特徴 |

### B級グルメ・軽食

| グルメ | 特徴 |
|---|---|
| **たこ判** | たこ焼きとお好み焼きをミックスした香川発のB級グルメ。大判焼きサイズでボリューム満点 |
| **かしわバター丼** | 香川発祥。鶏肉（かしわ）とバターを甘辛いタレで絡めたご当地丼 |
| **しょうゆ豆** | そら豆を炒ってから醤油・砂糖で味付けした香川の伝統的なおつまみ |

### スイーツ・特産品

| グルメ | 特徴 |
|---|---|
| **おいり** | もち米から作るカラフルな丸い和菓子。口に入れるとふわっと溶ける。結婚式の引き出物が由来 |
| **小豆島オリーブ** | 日本のオリーブ発祥の地。オリーブオイル、オリーブサイダー、オリーブソフトクリームなど多彩な商品 |
| **和三盆** | 讃岐の伝統的な高級砂糖。上品でまろやかな甘さの干菓子が人気のお土産 |

## 1泊2日モデルコース

### プランA：王道！うどん＆こんぴらさんコース

**1日目（高松〜琴平）**

1. 🍜 朝食：高松市内の人気うどん店でモーニングうどん
2. 🌿 栗林公園を散策
3. 🏯 丸亀城で日本一の石垣を見学
4. 🍗 昼食：丸亀市で骨付鳥
5. ⛩️ 金刀比羅宮の785段を参拝
6. 🍜 中野うどん学校でうどん手打ち体験
7. ♨️ 琴平温泉で疲れを癒す
8. 🏨 琴平エリアで宿泊

**2日目（西讃〜高松）**

1. 📸 父母ヶ浜で干潮のリフレクション写真撮影（干潮時間を事前チェック）
2. 💰 銭形砂絵を展望台から見学
3. 🌸 紫雲出山で瀬戸内海のパノラマを堪能
4. 🍜 昼食：帰り道にもう一杯讃岐うどん
5. 🛍️ 高松市内でお土産購入（和三盆・おいり）

### プランB：アートの島めぐりコース

**1日目（高松〜直島）**

1. ⛴️ 高松港からフェリーで直島へ（約50分）
2. 🎨 地中美術館でモネの睡蓮を鑑賞
3. 🟡 草間彌生の黄色いかぼちゃ＆赤かぼちゃ
4. 🏛️ ベネッセハウス ミュージアムを見学
5. 🍽️ 島内カフェで瀬戸内の食材を使ったランチ
6. 🏘️ 家プロジェクト（本村エリアのアート集落）を散策
7. 🏨 直島で宿泊 or フェリーで高松に戻り宿泊

**2日目（高松〜屋島）**

1. 🍜 朝食：讃岐うどん
2. ⛰️ 屋島で瀬戸内海の多島美を一望
3. 🏘️ 四国村で古民家とアートを楽しむ
4. 🌿 栗林公園を散策
5. 🛍️ 高松中央商店街でお土産＆食べ歩き

!!! tip "離島を巡るなら"
    小豆島や直島をじっくり巡りたい場合は2泊3日がおすすめ。フェリーの時刻表を事前に確認し、余裕を持ったスケジュールを組むこと。

## 無料キャンプ場

香川県は面積が小さいぶんキャンプ場の数は限られるが、無料・格安で利用できるスポットがある。

### おすすめ無料・格安キャンプ場

| キャンプ場 | エリア | 特徴 | 料金 |
|---|---|---|---|
| **大池オートキャンプ場** | 東かがわ市 | 大池のほとりにある自然豊かなキャンプ場。オートサイトあり | 格安 |
| **小豆島ふるさと村キャンプ場** | 小豆島 | 道の駅併設。オートキャンプ・テント泊可能。プールやテニスコートもある複合施設 | 格安 |
| **国営讃岐まんのう公園キャンプ場** | まんのう町 | 広大な国営公園内の整備されたキャンプ場。オートサイト・フリーサイトあり。設備充実 | 有料（要予約） |

!!! warning "キャンプ場利用時の注意"
    - 香川県は無料キャンプ場が少ないため、事前予約がおすすめ
    - 夏場は瀬戸内の穏やかな気候だが、日差しが強いため日よけ対策を
    - 最新の開設状況・料金は公式サイトで確認すること

## 車中泊スポット（道の駅・RVパーク）

瀬戸内の穏やかな気候と温泉施設が充実しており、快適な車中泊が楽しめる。

### おすすめ車中泊スポット

| スポット | 所在地 | 特徴 |
|---|---|---|
| **道の駅 たからだの里さいた** | 三豊市 | RVパーク併設。普通車199台の広い駐車場。「環の湯」温泉施設を完備 |
| **道の駅 香南楽湯** | 高松市 | 日帰り温泉を中心とした道の駅。広々とした湯船でリラックス。産直市場・レストラン併設 |
| **道の駅 小豆島オリーブ公園** | 小豆島 | オリーブ畑と瀬戸内海の絶景。「オートビレッジYOSHIDA」キャンプサイト併設 |
| **道の駅 ことひき** | 観音寺市 | 銭形砂絵の近く。琴弾公園内にあり海岸散歩も楽しめる |
| **道の駅 津田の松原** | さぬき市 | 白砂青松の津田の松原に隣接。海水浴場のすぐそばで夏場に人気 |

!!! tip "車中泊のマナー"
    道の駅は公共の休憩施設であり宿泊施設ではない。長期滞在・テント設営・外での調理は避け、エンジンのかけっぱなしにも配慮すること。RVパーク併設の「たからだの里さいた」は車中泊を正式に許可された施設なので安心。
