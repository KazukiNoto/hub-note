# 徳島県 旅行ガイド — 観光地・グルメ・モデルコース

## 概要

徳島県は四国の東部に位置し、豊かな自然・伝統文化・ご当地グルメが魅力の観光エリア。鳴門の渦潮や祖谷のかずら橋など、ダイナミックな自然スポットから、阿波おどりや藍染めなどの文化体験まで幅広く楽しめる。

---

## おすすめ観光スポット

### 鳴門エリア

| スポット | 概要 |
|---|---|
| **鳴門の渦潮** | 世界三大潮流のひとつ。直径最大30mの巨大な渦潮を観潮船や「渦の道」から間近で体感できる |
| **大塚国際美術館** | 世界の名画を陶板で原寸大再現した美術館。圧倒的なスケールで1日楽しめる |
| **鳴門公園・千畳敷展望台** | 鳴門海峡を一望できる絶景スポット |

### 祖谷・大歩危エリア

| スポット | 概要 |
|---|---|
| **祖谷のかずら橋** | シラクチカズラで編まれた吊り橋。日本三奇橋のひとつでスリル満点 |
| **大歩危・小歩危** | 吉野川の激流が生んだ約8kmの渓谷美。ラフティングや遊覧船も楽しめる |
| **奥祖谷二重かずら橋** | 「男橋」「女橋」の2本が架かる秘境スポット |
| **落合集落** | 山の急斜面に広がる伝統的集落。国の重要伝統的建造物群保存地区 |

### 徳島市エリア

| スポット | 概要 |
|---|---|
| **阿波おどり会館** | 年間を通じて本格的な阿波おどりを鑑賞・体験できる施設 |
| **眉山（びざん）** | 徳島市のシンボル。ロープウェイで山頂へ登ると市街地と海を一望できる |
| **ひょうたん島クルーズ** | 徳島市中心部を囲む川を周遊する約30分のクルーズ |

### 西部エリア

| スポット | 概要 |
|---|---|
| **脇町うだつの町並み** | 江戸〜明治時代の商家が立ち並ぶ歴史的な町並み。「うだつが上がる」の語源 |
| **阿波の土柱** | 世界三大土柱のひとつ。自然が生んだ奇観 |
| **藍の館** | 徳島が誇る藍染め文化を学び、藍染め体験もできる |

## おすすめご当地グルメ

### 麺・主食

| グルメ | 特徴 |
|---|---|
| **徳島ラーメン** | 白・黄・茶の3種類のスープが特徴。豚バラ肉トッピングに生卵を落とし、白ご飯と一緒に食べるのが徳島流 |
| **たらいうどん** | たらいに入った素朴なうどん。祖谷地方の名物 |
| **そば米汁** | 祖谷地方の郷土料理。そば米と鶏肉・野菜の素朴な汁物 |

### 肉・海鮮

| グルメ | 特徴 |
|---|---|
| **鳴門鯛** | 激しい潮流で育った真鯛。身が引き締まりコリコリの食感 |
| **阿波尾鶏（あわおどり）** | 徳島のブランド地鶏。旨み豊富で適度な歯応え |
| **阿波牛** | 徳島県産の黒毛和牛。きめ細かな霜降りが特徴 |

### B級グルメ・軽食

| グルメ | 特徴 |
|---|---|
| **フィッシュカツ** | 魚のすり身にカレー粉・唐辛子を加えて揚げた徳島のソウルフード |
| **でこまわし** | じゃがいも・豆腐・こんにゃくを串に刺し味噌だれで焼く祖谷の田楽 |
| **まめ玉** | お好み焼きに甘い金時豆をトッピングした独特のご当地グルメ |
| **とくしまバーガー** | 阿波牛・阿波尾鶏・鳴門鯛など地元食材を使ったご当地バーガー |

### 特産品

| 特産品 | 特徴 |
|---|---|
| **すだち** | 全国生産量の約98%を占める徳島の代表的柑橘。ビタミンC・クエン酸はレモン以上 |
| **鳴門金時** | ホクホクで甘みの強いさつまいも。スイーツにも多く使われる |
| **鳴門わかめ** | 鳴門海峡の激しい潮流で育った肉厚でシャキシャキのわかめ |

## 1泊2日モデルコース

### プランA：自然と歴史を満喫コース

**1日目（鳴門〜徳島市）**

1. 🌊 「鳴門の渦潮」観潮船で渦潮を体感
2. 🖼️ 大塚国際美術館で世界の名画を鑑賞
3. 🍜 昼食：鳴門鯛の料理を堪能
4. 🏯 藍の館で藍染め体験
5. 💃 阿波おどり会館で阿波おどり鑑賞
6. 🍜 夕食：徳島ラーメン
7. 🏨 徳島市内で宿泊

**2日目（祖谷〜脇町）**

1. 🌉 祖谷のかずら橋を渡る
2. 🍢 昼食：でこまわし＆そば米汁
3. 🏔️ 大歩危の遊覧船で渓谷美を楽しむ
4. 🏘️ 脇町うだつの町並みを散策
5. 🗿 阿波の土柱を見学

### プランB：体験とグルメを楽しむコース

**1日目（徳島市）**

1. 🎨 藍の館で藍染め体験
2. 🍜 昼食：徳島ラーメン
3. 🚢 ひょうたん島クルーズ
4. 💃 阿波おどり会館で踊り体験
5. 🌄 眉山ロープウェイで夕景を楽しむ
6. 🍗 夕食：阿波尾鶏料理
7. 🏨 徳島市内で宿泊

**2日目（鳴門）**

1. 🌊 「鳴門の渦潮」観潮船
2. 🌁 渦の道・千畳敷展望台
3. 🍛 昼食：フィッシュカツ＆とくしまバーガー
4. 🖼️ 大塚国際美術館
