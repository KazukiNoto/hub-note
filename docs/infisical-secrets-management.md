# Infisical — 開発者・AIエージェント向けモダンシークレット管理プラットフォーム

**Source**: <https://infisical.com>

## 概要

[Infisical](https://infisical.com) は、APIキー・データベース認証情報・設定値などのシークレット（機密情報）を一元管理するためのオープンソースプラットフォームです。ローカル開発から本番環境まで、開発ライフサイクル全体を通じてシークレットを安全に管理できます。

MITライセンスのもとオープンソースとして公開されており、**セルフホスト**と**クラウド（Infisical Cloud）**の両方に対応しています。

---

## 主な機能

### シークレット管理

- APIキー・DBパスワード・環境変数などを一元管理
- シークレットのバージョン管理・ポイントインタイムリカバリー
- シークレット参照（Secret Referencing）による再利用
- シークレットスキャン（漏洩検知）
- 環境ごとの分離（開発 / ステージング / 本番）

### 動的シークレット（Dynamic Secrets）

オンデマンドでクライアントごとにユニークなシークレットを生成する機能。静的なパスワードより大幅にセキュリティを向上できます。

### PKI・証明書管理

- 内部・外部 PKI を横断した証明書ライフサイクル管理
- 証明書の自動発行・更新により有効期限切れリスクを排除

### アクセス制御・監査

- 人間・マシン ID に対するきめ細かな権限管理（RBAC）
- 一時的な昇格アクセス（Temporary Elevated Access）と承認ワークフロー
- 外部コントラクターへの期限付きアクセス付与
- シークレット変更・ローテーション時の通知
- 監査ログ（プランに応じて最大 90 日以上）

### AIエージェントセキュリティ

AIエージェントがツールや外部システムにアクセスする際のガバナンスを提供。MCP（Model Context Protocol）エンドポイント全体にわたって認証・ポリシー適用・可視化を一元化します。

## 主なインテグレーション

Infisical は幅広いプラットフォームへのシークレット同期（Secrets Syncs）をサポートしています。

| カテゴリ | 対応サービス例 |
|---|---|
| クラウド | AWS Secrets Manager、GCP、Azure |
| CI/CD | GitHub Actions、GitLab CI |
| コード管理 | GitHub、GitLab |
| インフラ | Kubernetes、Terraform |
| デプロイ | Vercel、Netlify |
| ローカル開発 | CLI、SDK |

## まとめ

Infisical はシークレット管理の課題をワンストップで解決するプラットフォームです。オープンソースで透明性が高く、セルフホストにも対応しているため、コンプライアンスやデータ主権の要件が厳しい組織でも採用しやすい選択肢です。AIエージェントのセキュリティ管理機能を備えている点も、2024〜2026年のAI普及トレンドに合わせた特徴といえます。
