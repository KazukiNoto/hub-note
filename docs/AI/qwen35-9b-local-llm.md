# Qwen 3.5-9B — 9Bで120B超え、アリババの小型最強ローカルLLM

**Source**: <https://huggingface.co/Qwen/Qwen3.5-9B>

## 概要

**Qwen 3.5-9B** は、Alibaba Cloud の Qwen チームが2026年3月2日に公開した小型言語モデル。わずか9Bパラメータながら、OpenAIの120Bモデルを超えるベンチマーク性能を叩き出し、「ローカルAIの常識を変えた」と話題に。スマートフォンやノートPCで完全オフライン動作し、エンタープライズグレードのAI性能を実現する。

---

## 主なスペック

| 項目 | 内容 |
|---|---|
| **開発元** | Alibaba Cloud Qwen チーム |
| **公開日** | 2026年3月2日 |
| **パラメータ数** | 9B（90億） |

| **マルチモーダル** | テキスト・画像・動画（字幕付き）に対応 |
| **ライセンス** | Apache 2.0（オープンソース） |

---

## 革新的なアーキテクチャ

Qwen3.5-9Bの最大の技術的ブレークスルーは、**Gated Delta Networks（DeltaNet）** と **Sparse Mixture-of-Experts（MoE）** を組み合わせたハイブリッドアーキテクチャにある。

- 線形注意とソフトマックス注意を **3:1** の比率で採用
- メモリ消費と計算コストを大幅に削減
- 小型ながら大規模モデルに匹敵する推論能力を実現

## ローカル実行方法

### 必要スペック

| 環境 | 推奨 |
|---|---|
| **最低メモリ** | 6GB VRAM（Q4量子化時） |
| **推奨メモリ** | 12GB以上（フル精度） |
| **Mac** | 16GB ユニファイドメモリ |
| **GPU** | RTX 4070以上 |

### Ollamaで実行（最も簡単）

```bash
# インストール後、ワンコマンドで起動
ollama run qwen3.5:9b
```

### LM Studioで実行

1. LM Studioをダウンロード・インストール
2. 検索バーで「Qwen3.5-9B」を検索
3. Q4_K_M または Unsloth UD-Q4_K_XL を選択してダウンロード
4. チャット画面から即座に利用開始

## まとめ

Qwen 3.5-9Bは、小型モデルの限界を大きく押し広げた革新的なLLM。ハイブリッドアーキテクチャにより、従来の3倍以上のパラメータを持つモデルを超える性能を実現し、一般的なノートPCやデスクトップで完全オフライン動作する。日本語性能も高く、ローカルLLMの導入を考えている人にとって最も実用的な選択肢のひとつ。
