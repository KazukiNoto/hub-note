# WriteBack-RAG: 企業ナレッジの自動整理と検索精度向上

**Source**: https://zenn.dev/knowledgesense/articles/5a50d06fce072d

## 概要

WriteBack-RAGは、ユーザーからの問い合わせに答える過程で得た知見を自動的に整理し、企業ナレッジベースとして「書き戻す」革新的なRAG手法です。大企業の社内データに散在する重要情報に対応し、次回以降の検索精度向上を実現します。

## 背景と課題

大企業の社内データは以下のような特徴があります：

- 重要な情報が多くのドキュメント・システムに散在している
- 通常のRAG（Retrieval Augmented Generation）では対応が難しい
- 既存のナレッジベースは非効率な構成になりがち

## WriteBack-RAGの実装プロセス

1. **有用なサンプル選別** — ユーザー問い合わせから得られた回答を評価
2. **重要文書特定** — 関連する複数の文書を識別
3. **文書蒸留** — LLMを使用して複数の文書を1つのコンパクトな文書に統合
4. **ナレッジベースへの書き戻し** — 新たに蒸留されたドキュメントをナレッジベースに追加

## 実績

全48個の設定条件で性能が向上しました：

| タスク | 改善率 |
|--------|--------|
| 全体平均 | 2.14% |
| 事実検証タスク | 4.79% |
| オープンドメインQA | 3.01% |
