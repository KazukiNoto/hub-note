# AppFlowy — オープンソースのNotion代替コラボレーションワークスペース

**Source**: <https://github.com/AppFlowy-IO/AppFlowy>

## 概要

AppFlowy は「データの主権を失わずに生産性を高められるAIコラボレーションワークスペース」として開発されているオープンソースプロジェクトです。Notionの代替として位置づけられており、データプライバシー・ネイティブなクロスプラットフォーム体験・コミュニティ主導の拡張性を三大方針として掲げています。

GitHubスター数は **69,600以上**、フォーク数は **5,100以上** と非常に活発なプロジェクトです。

## 主な機能

- **カンバンボード** — タスク管理をビジュアルに
- **データベースビュー** — プロジェクト・構造化データの管理
- **ドキュメント / サイト作成** — 美しいドキュメントを作成
- **AI統合** — 生産性を高めるAIアシスタント機能
- **テンプレート** — すばやくプロジェクトを立ち上げ
- **クロスプラットフォーム** — デスクトップ・モバイル両対応

## 技術スタック

| 言語 / フレームワーク | 割合 | 用途 |
|---|---|---|
| Dart / Flutter | 73.8% | フロントエンド・UI |
| Rust | 24.1% | バックエンド・コアロジック |
| C++, Shell, CMake, HTML | 残り | ビルド・補助 |

## 対応プラットフォーム

- **デスクトップ**: macOS / Windows / Linux（FlatHub・Snapcraft・SourceForge経由でも配布）
- **モバイル**: iOS（App Store）/ Android 10以上（Google Play）
- **セルフホスティング**: ソースからのビルドも可能

## まとめ

AppFlowy は、Notionのような強力なワークスペース機能をオープンソースかつセルフホスト可能な形で提供するプロジェクトです。FlutterとRustという現代的な技術スタックを採用し、データをクラウドサービスに預けたくないユーザーや、ツールを自由にカスタマイズしたいチームに適した選択肢です。活発なコミュニティ（Discord・Reddit・フォーラム）も強みの一つです。
