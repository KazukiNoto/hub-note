# 富士通、国内初のソースコード解析・設計書自動生成AIサービスを提供開始

**Source**: <https://aismiley.co.jp/ai_news/fujitsu-sourcecode-ai-offer/>

## 概要

富士通は2026年3月30日、レガシーシステムのソースコード解析と設計書自動生成を行うSaaSサービス「**Fujitsu Application Transform powered by Fujitsu Kozuchi**」の国内提供を開始しました。COBOL等の既存ソースコードを生成AIで解析し、設計書を自動生成する国内初のサービスです。

## 背景・課題

- 長年運用されてきたレガシーシステムは設計書が更新されず、仕様が**ブラックボックス化**している
- COBOL言語は記述方法が多様で、専門有識者による解析が困難
- モダナイゼーション（システム近代化）推進の妨げになっている

## 主なポイント

- **対象言語**: COBOL等の既存ソースコード
- **技術基盤**: ナレッジグラフ拡張RAG（Retrieval-Augmented Generation）による高精度分析
- **提供形態**: SaaS（Software as a Service）

## 導入効果

| 指標 | 効果 |
|------|------|
| 作業時間 | 約 **1/30** に短縮 |
| 設計書の網羅性 | **95%** 向上 |
| 可読性 | **60%** 向上 |

!!! note "ポイント"
    有識者がいない環境でも、従来の約1/30の時間で設計書を生成できるとされています。

## 今後のロードマップ

サービスは順次機能を追加予定：

1. **導入支援**
2. **リビルド機能** — システムの再構築支援
3. **リライト機能** — コードの書き換え支援
4. **運用・保守機能** — 継続的なシステム管理支援

## まとめ

生成AIとRAG技術を活用し、長年のレガシーシステム問題（ブラックボックス化・有識者不足）を解決するソリューション。日本企業のDX推進・モダナイゼーション加速に貢献することが期待されます。
