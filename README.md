# hub-note

Zensicalを使って雑記をGitHub Pagesで公開する実験用プロジェクトです。

## Zensical

https://zensical.org

### Zensicalとは
**Zensical**は、Material for MkDocsの開発チームがRustで構築した次世代の高速静的サイトジェネレーターです。

Markdown形式で簡単に洗練されたドキュメントやブログを作成でき、大規模サイトでも瞬時に生成できるパフォーマンスを備えています。

開発ワークフローとの親和性が高く、ドキュメント管理の効率化に特化した最新ツールです。

### インストール
``` bash
pip install zensical
```

### 初期設定

``` bash
zensical new .
```
